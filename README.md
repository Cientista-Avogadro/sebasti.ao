# Sebastião de Sousa Moniz | Portfolio

Personal portfolio website for Sebastião de Sousa Moniz - Senior Software Engineer and System Analyst from Angola.

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Deployment**: Vercel

## Features

- Responsive design with dark theme
- Hero section with typewriter effect
- Timeline view for experience
- Filterable projects, courses, and gallery
- SEO optimized (OpenGraph, JSON-LD, sitemap)
- Lightbox gallery for photos and videos

## Pages

- `/` - Homepage with all sections
- `/projects` - All projects with filters
- `/courses` - Certifications and courses
- `/gallery` - Event photos and videos
- `/articles` - LinkedIn articles

## Getting Started

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Production build
npm run build

# Start production server
npm start
```

## Project Structure

```
src/

│   └── sections/     # Page sections
```
